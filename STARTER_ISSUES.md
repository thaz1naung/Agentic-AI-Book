# Starter Issues

These are suggested issues the maintainer can manually open in GitHub.

## 1. Improve Agent Loop Diagram

**Description:** Add or improve a beginner-friendly diagram that shows observe, think, act, and verify steps in an agent loop.

**Suggested label:** `diagram`

**Difficulty:** Beginner / Intermediate

## 2. Add MCP Beginner Explanation

**Description:** Propose a simple explanation of MCP using client, server, tools, and permissions. Keep it beginner-friendly.

**Suggested label:** `chapter suggestion`

**Difficulty:** Intermediate

## 3. Add DevOps Automation Case Study

**Description:** Suggest a safe DevOps automation case study with clear lab-only boundaries and no production-destructive steps.

**Suggested label:** `roadmap`

**Difficulty:** Intermediate

## 4. Review Burmese/English Terminology Consistency

**Description:** Review repeated technical terms and suggest where English should be kept or where Burmese explanation should be added.

**Suggested label:** `terminology`

**Difficulty:** Beginner

## 5. Add Context Engineering Diagram

**Description:** Suggest a diagram showing user message, system prompt, memory, retrieved documents, and tool results as model context.

**Suggested label:** `diagram`

**Difficulty:** Beginner / Intermediate

## 6. Add RAG Memory Example

**Description:** Suggest a small example showing retrieval, memory, and final answer flow without overcomplicating the implementation.

**Suggested label:** `documentation`

**Difficulty:** Intermediate

## 7. Add LangGraph Practical Example

**Description:** Propose a practical LangGraph example that explains nodes, edges, state, and retry behavior.

**Suggested label:** `technical correction`

**Difficulty:** Intermediate / Advanced

## 8. Add Safe Agent Tool-Calling Example

**Description:** Suggest a safe tool-calling example with permission limits, validation, and human approval for risky actions.

**Suggested label:** `security`

**Difficulty:** Intermediate

## 9. Improve Repository README Links

**Description:** Check README links and suggest improvements if any link is missing, confusing, or hard to find.

**Suggested label:** `good first issue`

**Difficulty:** Beginner
