# Contributing To Agentic AI Book

Thanks for wanting to help. This repo is a book project, so contributions should be careful and small.

## What You Can Help With

Good contributions include:

- typo fixes
- wording suggestions
- diagrams
- terminology consistency
- beginner explanation improvements
- issue discussions
- example corrections
- broken link reports
- glossary term suggestions

## Important Manuscript Rule

Manuscript/chapter rewrites should be proposed through issues first.

Please do not directly rewrite chapter sections in a pull request unless the maintainer has approved that approach. The chapters are author-owned writing, and the Burmese + English voice matters.

Small typo fixes are okay. Large prose changes need discussion first.

## Contribution Flow

1. Fork the repo.
2. Create a branch.
3. Make a small focused change.
4. Open a pull request.
5. Explain what changed and why.

Example branch names:

```text
fix/readme-link
docs/add-glossary-term
diagram/agent-loop-suggestion
```

## Pull Request Tips

- Keep one PR focused on one topic.
- Link a related issue if there is one.
- Do not add fake links, fake sources, or claims you cannot support.
- For technical corrections, explain the source of the correction.
- For chapter wording suggestions, preserve the author's voice as much as possible.

## Good First Contributions

- Fix spelling or punctuation.
- Suggest a missing glossary term.
- Report a confusing paragraph through an issue.
- Propose a diagram in an issue before making a big file change.
- Improve repository support docs like README or CONTRIBUTING.
